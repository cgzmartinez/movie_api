Documentation Page

This project is a REST API, an architected database built using JavaScript, Node.js, Express, and MongoDB. This REST API, used for the movie app "myFlix", will be accessed via commonly used HTTP methods like GET and POST. Similar methods (CRUD) will be used to retrieve data from your database and store that data in a non-relational way.

Returns a list of all movies to the user "/movies" none GET

Allows new users to register "/users" none POST
Example: app.post("/users", (req,res) =>{})		 A JSON object holding data about new user that was added
Example: { ID: , Username: , Password: , Email: , }, 		Required: /users
Example: localhost:8080/users/ { id: 1, name: "Mary", favoriteMovies: [] };
Allows users to update their user info "/users/:username" :id PUT
Example: app.put("/users:username", (req,res) =>{}) 		A JSON object holding the updated user data
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }, 		Required: /users, :username
Example: localhost:8080/users/Mary
Allows users to add a movie to their list of favorites "/users/:username/movies/:movieID" :username,:movieTitle POST
Example: app.post("/users/:username/movies/:movieID", (req,res) =>{}) 		A text message indicating that the movie has been added Required: :id, :movieTitle
Example: localhost:8080/users/1/movies/John%20Wick
Returns Data About a Director by Name "/movies/director/:directorName" :name GET
Example: app.get("/movies/director/:directorName", (req,res) =>{}) 		A JSON object holding data about a director
Example: Director: { Name: "Denis Villeneuve", Bio: "" }, 		Required: :directorName
Example: localhost:8080/movies/director/Denis%20Villeneuve
